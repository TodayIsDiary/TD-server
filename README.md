## 프로젝트 소개

### 서비스 이름 : 🌃 하루의 끝 (Today_IS_Diary) 🌃

> 하루를 끝내며 쓰는 일기들을 쓰고 다른사람들과 자신의 하루를 공유할 수 있는 커뮤니티형 서비스

- **부제**: 사용자의 상황 맞춤형 소통 및 공유 플랫폼

- **타겟층**: 하루가 끝나기전 일기를 쓰시는 분, 새벽감성을 좋아 하시는 분

- **서비스 주요 기능**<br/>
  1️⃣ 오늘의 감정일기 쓰기<br/>
  2️⃣ 카테고리별 일기 리스트<br/>
  3️⃣ 부적절한 일기&사용자&댓글 신고<br/>
